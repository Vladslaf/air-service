## air-service

### Technologies used
- Spring Boot 2.3
- Swagger
- Lombok
- REST API
- JSON
- MySQL

### How to run
1. Clone
2. Enable Lombok annotation
3. Configure DB connection (login,password)
4. Run
5. Some initial data provided in data/data.sql along with Postman collection in source folder or Swagger
```
http://localhost:8080/swagger-ui/#/
```
